# input

> input 标签规定了用户可以在其中输入数据的输入字段， input 标签在 form 标签中使用，用来声明允许用户输入数据的 input 控件。

语法：

```html
<form>
    First name： <input type="text" name="first_name"><br>
    Last name： <input type="text" name="last_name">
</form>
```

效果：

<form>
    First name： <input type="text" name="first_name"><br>
    Last name： <input type="text" name="last_name">
</form>
