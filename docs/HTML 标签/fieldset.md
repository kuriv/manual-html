# fieldset

> fieldset 标签可以将表单内的相关元素分组。

语法：

```html
<form>
    <fieldset>
        <legend>个人信息:</legend>
        姓名: <input type="text" size="30"><br>
        邮箱: <input type="text" size="30">
    </fieldset>
</form>
```

效果：

<form>
    <fieldset>
        <legend>个人信息:</legend>
        姓名: <input type="text" size="30"><br>
        邮箱: <input type="text" size="30">
    </fieldset>
</form>
