# bdi

> bdi 标签允许设置一段文本，使其脱离其父元素的文本方向设置，在发布用户评论或其他无法完全控制的内容时，该标签很有用。

语法：

```html
<ul>
    <li>Username <bdi>Bill</bdi> ： 80 points</li>
    <li>Username <bdi>Steve</bdi> ： 78 points</li>
</ul>
```

效果：

<ul>
    <li>Username <bdi>Bill</bdi> ： 80 points</li>
    <li>Username <bdi>Steve</bdi> ： 78 points</li>
</ul>
