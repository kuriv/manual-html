# dialog

> dialog 标签定义对话框或窗口。

语法：

```html
<table>
    <tr>
        <th>January <dialog open>This is an open dialog window!</dialog></th>
        <th>February</th>
        <th>March</th>
    </tr>
    <tr>
        <td>31</td>
        <td>28</td>
        <td>31</td>
    </tr>
</table>
```

效果：

<table>
    <tr>
        <th>January <dialog open>This is an open dialog window!</dialog></th>
        <th>February</th>
        <th>March</th>
    </tr>
    <tr>
        <td>31</td>
        <td>28</td>
        <td>31</td>
    </tr>
</table>
