# style

> style 标签包含了 HTML 文档的详细样式，默认情况下在该元素内写入的样式指令将被认为是 CSS 。

语法：

```html
<!DOCTYPE html>
<html lang="zh-cmn-Hans">
    <head>
        <style type="text/css">
            h1 {
                color: red;
            }
            p {
                color: blue;
            }
        </style>
    </head>
    <body>
        <h1>A heading</h1>
        <p>A paragraph.</p>
    </body>
</html>
```
