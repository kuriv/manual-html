# track

> track 标签为媒体元素规定外部文本轨道。

语法：

```html
<video width="400" controls>
    <source src="video.mp4" type="video/mp4">
    <track src="subtitles.vtt" kind="subtitles">
</video>
```

效果：

<video width="400" controls>
    <source src="video.mp4" type="video/mp4">
    <track src="subtitles.vtt" kind="subtitles">
</video>
