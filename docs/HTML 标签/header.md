# header

> header 标签定义文档或者文档的一部分区域的页眉。

语法：

```html
<article>
    <header>
        <h1>GitHub</h1>
    </header>
    <p>GitHub is a development platform inspired by the way you work.</p>
</article>
```

效果：

<article>
    <header>
        <h1>GitHub</h1>
    </header>
    <p>GitHub is a development platform inspired by the way you work.</p>
</article>
