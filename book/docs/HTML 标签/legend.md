# legend

> legend 标签为 fieldset 标签定义标题。

语法：

```html
<form>
    <fieldset>
        <legend>个人信息:</legend>
        姓名: <input type="text" size="30"><br>
        邮箱: <input type="text" size="30">
    </fieldset>
</form>
```

效果：

<form>
    <fieldset>
        <legend>个人信息:</legend>
        姓名: <input type="text" size="30"><br>
        邮箱: <input type="text" size="30">
    </fieldset>
</form>
