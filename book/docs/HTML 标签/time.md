# time

> time 标签用来表示 HTML 网页中的日期和时间，让搜索引擎等程序更容易的提取这些信息。

语法：

```html
<p>We open at <time>10:00</time> every morning.</p>
<p>I have a date on <time datetime="2020-02-14">Valentines day</time>.</p>
```

效果：

<p>We open at <time>10:00</time> every morning.</p>
<p>I have a date on <time datetime="2020-02-14">Valentines day</time>.</p>
