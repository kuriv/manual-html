# summary

> summary 标签元素作为一个 details 元素的标题，该标题可以包含详细的信息，但是默认情况下不显示，需要单击才能显示详细信息。

语法：

```html
<details>
    <summary>Copyright 2020.</summary>
    <p> - by Kuriv Vesti. All Rights Reserved.</p>
</details>
```

效果：

<details>
    <summary>Copyright 2020.</summary>
    <p> - by Kuriv Vesti. All Rights Reserved.</p>
</details>
