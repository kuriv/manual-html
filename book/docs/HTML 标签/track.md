# track

> track 标签为媒体元素规定外部文本轨道。

语法：

```html
<video controls width="400">
    <source src="video.mp4" type="video/mp4">
    <track kind="subtitles" src="subtitles.vtt">
</video>
```

效果：

<video controls width="400">
    <source src="video.mp4" type="video/mp4">
    <track kind="subtitles" src="subtitles.vtt">
</video>
