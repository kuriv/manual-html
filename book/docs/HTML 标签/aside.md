# aside

> aside 标签为语义化标签， aside 标签在 article 标签之外使用作为页面或站点全局的附属信息部分，并且内容应该与附近的内容相关。

语法：

```html
<h1>GitHub</h1>
<aside>
    <p>GitHub is a development platform inspired by the way you work.</p>
</aside>
```

效果：

<h1>GitHub</h1>
<aside>
    <p>GitHub is a development platform inspired by the way you work.</p>
</aside>
