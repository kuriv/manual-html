# figcaption

> figcaption 标签为 figure 标签定义标题。

语法：

```html
<figure>
    <img src="https://github.com/fluidicon.png" alt="GitHub">
    <figcaption>GitHub</figcaption>
</figure>
```

效果：

<figure>
    <img src="https://github.com/fluidicon.png" alt="GitHub">
    <figcaption>GitHub</figcaption>
</figure>
